变量删除_替换_替代

基础知识

  • ${#value} 变量的字符个数

特殊变量:

  • $#, 程序后接的参数个数
  • $@, "$1"、"$2"、"$3"、"$4" Save all the parameters.
  • $0 The name of current program.
  • $*, 与$@类似
  • $$, 本shell的pid
  • $! Last program's PID.
  • $? 上个命令的回传码 Last command or function's return value.

删除

  • ${value#pattern}

  • ${value##pattern}

    去掉value中与pattern相匹配的部分,条件是value的开头与pattern相匹配

    #与##的区别在于一个是最短匹配模式,一个是最长匹配模式.

  • ${value%pattern}

  • ${value%%pattern}

    从value的尾部于pattern相匹配,%与%%的区别与#与##一样

替换

  • ${value/pattern/string}
  • ${value//pattern/string}

    进行变量内容的替换,把与pattern匹配的部分替换为string的内容.
    /与//的区别：// 是全部替换

替代

目标：由${str} 和字符串expr 得到$
条件：

  • str:

    str没设置或为空字符串

  • str

    str没设置

用法

  • var=${str:-expr}

    当条件为真时（变量未定义或者值为空时）,返回值为expr的内容,否则返回变量str的值.

  • var=${str-expr}

当条件为真时（变量未定义）,返回值为expr的内容,否则返回变量str的值.

总结
  • -与=：=是在-的基础上连同str一起更改
  • -与+：正好相反
  • -与？：条件为真时，expr输出到stderr，其他不变 ？其实是当旧变量不存在时，给出警告
