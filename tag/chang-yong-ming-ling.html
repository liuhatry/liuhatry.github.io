常用命令

后台运行

command & 退出终端后就不运行了

nohup command args & (no hang up)退出终端，也继续运行

原程序的的标准输出被自动改向到当前目录下的nohup.out文件，起到了log的作用， 重定向：nohup command > myout.file 2>&1 &

jobs
bg
fg

数据流冲定向

  • 标准输出：代码为1

    > 或1> 覆盖
    >> 或1>> 累加

  • 标准错误输出：代码为2

    2>
    2>>

  • 标准输入：代码为0

    <
    <<

  • 双向重定向：tee

    last | tee last.list | cut -d " " -f1

  • 常用技巧

    将错误丢弃：2> /dev/null
    将正确与错误同时写入文件：&> file 或 > file 2>&1
    用cat创建文件：cat > filename 输入文字后用Ctrl+D结束输入
